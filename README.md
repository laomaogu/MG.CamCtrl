### 创作不易，请:tw-2b50:支持

## 介绍

c# 工业相机库，含海康(HIK)、大恒、巴斯勒(Basler)等相机的常用功能。
底层采用回调+信号量模式封装 ，最大程度减小线程资源，提高采图效率。
