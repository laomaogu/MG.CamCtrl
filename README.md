# C#  MG.CamCtrl 相机库（开源） 海康 大恒

## 介绍

c# 相机库，含海康、大恒品牌2D相机的常用功能。
底层采用回调+信号量模式封装 ，最大程度减小线程资源，提高采图效率。

 
  **功能持续完善中**
